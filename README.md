# 🏦 Bradesco Web

> Simulação de internet banking do Bradesco desenvolvida como projeto acadêmico e de portfólio, com foco em experiência do usuário, boas práticas de desenvolvimento front-end e trabalho colaborativo com Git e GitHub.

---

## 📸 Visão Geral

O projeto simula as principais funcionalidades de um internet banking moderno, com interface limpa e navegação intuitiva inspirada nas soluções digitais do Bradesco.

---

## 🚀 Funcionalidades

| Página | Descrição |
|--------|-----------|
| 🏠 **Início** | Tela principal com saldo, atalhos rápidos e últimas transações |
| 📊 **Dashboard** | Painel completo com gráficos de receitas x despesas e gastos por categoria |
| ⚡ **PIX** | Simulação de transferência via PIX |
| 💸 **Transferir** | Transferência por etapas (TED, DOC, entre contas e agendada) |
| 📄 **Extrato** | Histórico de transações com filtros por período e tipo |
| 📱 **Recarga de Celular** | Recarga para as principais operadoras |
| 🚌 **Bilhete Único** | Recarga SPTrans com tipos Comum, Estudante, Mensal e TOP |
| 💳 **Cartões** | Gerenciamento de cartões de crédito e débito |
| 💰 **Criptomoedas** | Painel de acompanhamento e simulação de criptoativos |

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica das páginas
- **CSS3** — Estilização e layout responsivo
- **JavaScript** — Interatividade e lógica das funcionalidades
- **Font Awesome 6** — Ícones
- **Chart.js** — Gráficos do dashboard
- **Google Fonts** — Tipografia (Inter, Sora)
- **Git & GitHub** — Versionamento e colaboração em equipe

## ▶️ Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/matheuscarvalhotech/bradesco-web.git
```

2. Acesse a pasta do projeto:
```bash
cd bradesco-web
```

3. Abra o `index.html` no navegador ou use a extensão **Live Server** no VS Code.
